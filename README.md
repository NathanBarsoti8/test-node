# Node
> A simple CRUD in NodeJS with MongoDB and comunicate with HgBrasil Weather API

## Instalação e Configuração para Desenvolvimento

```sh
npm install
```

## Para rodar o projeto

```sh
npm run dev
```

## Para rodar os testes

```sh
npm test
```
